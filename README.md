# Chef Node
[![Circle CI](https://circleci.com/gh/usemarkup/chef-node.svg?style=svg)](https://circleci.com/gh/usemarkup/chef-node)

Installs node from https://rpm.nodesource.com/pub_5.x/el/6/x86_64/ using the EPEL RPMs

## Support

- CentOS 6.7
- CentOS 6.8
- CentOS 7.2
- Debian
- Ubuntu

## Usage

0.10 Release
```
  "default_attributes": {
    "node": {
      "major_version": "0.10",
      "version": "0.10.42.1"
    }
  }
```
