# Chef Node
[![Build Status](https://travis-ci.org/usemarkup/chef-node.svg?branch=master)](https://travis-ci.org/usemarkup/chef-node)

Installs node from https://github.com/nodesource/distributions for CentOS / Debian

## Support

- CentOS 6.x
- CentOS 7.x
- Debian

### Chef Support (tested)

- Chef 12.7+
- Chef 13.1
